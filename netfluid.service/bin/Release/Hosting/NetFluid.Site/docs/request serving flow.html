NetFluid run inside a security sandbox, anything not strictly specified as visible is unreachable by the outside

There are 6 ways to expose methods and files to clients, checked in this order.

  • Controllers: Lambda functions invoked any time the server receive a request
  • RegexRoutes: The method is called if the current Uri match the assigned regex, any method parameter is taken from regex groups
  • ParametrizedRoutes: Assign a variable Uri to a method, any method parameter is taken from Uri parts
  • Routes: Assign a fixed Uri to a method, any method parameter is taken from GET or POST data
  • Public folders: Assign a variable Uri to a physical folder, any files in folder is public and downloadable by the client.
  • StatusCode handlers: Assign a method to a specified status code.
