Controllers

Controllers are lambda function invoked everytime the server recieve a request.
Controllers can be executed under certain conditions or always and if they return a value, the context is close otherwise they are executed in background and completly insible by the user

How does it work

This simple page is your "website", everytime a client do a request the DataCollector in ./Analyics/DataCollector is executed.
Because the DataCollector controller doesn't close the context, the execution go on and this page is executed.
In analytics you can see the collected data.
