Controllers

Controllers are controls instantiated and executed everytime the server recieve a request.
Controllers are implemented by the FluidController class, like in this example, but if you need something smaller like executing a single action you can use the Engine.SetSmallController(Action)

How does it work

This simple page is your "website", everytime a client do a request the DataCollector in ./Analyics/DataCollector is executed.
Because the DataCollector controller doesn't close the context, the execution go on and this page is executed.
In analytics you can see the collected data.
