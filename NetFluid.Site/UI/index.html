What is it

Born with the name of the CSP (C Sharp Pages) Netfluid is a framework to develop custom application server and web application as standalone application or library.

Licensing

NetFluid is release under AGPL license ("Affero")

In short term: you can use netfluid without any kind of warranty and any kind of charge if the result of your work is open source and under the same license. If want more assistance or keep the logic of your web application confidential you need to pay me.

Legals and copyrights

Netfluid contains parts of the following software: DNS.NET Resolver by Alphons van der Heijden, MimeKit by Jeffrey Stedfast and JINT by Sébastien Ros.
I would like to express my sincere gratitude and gratitude and appreciation to all of them.

Not just HTTP

Netfluid is built to mix different kind of servers into a single high-fuctionality web application without any kind of external binding.
Built inside the framework you will find:

  • HTTP/S server
  • SMTP server
  • DSN server
  • FTP server

How to use it

Netfluid requires Microsoft .net framework © 4.5 or Mono © 3.4

  • Install prerequisites
  • download the Netfluid framework
  • Make a new project referencing NetFluid.dll
  • Develop and debug your web application from your favorite C# IDE
The old and hated Netfluid Compiler doesn't exists anymore, now you can develop and run directly from Visual Studio
